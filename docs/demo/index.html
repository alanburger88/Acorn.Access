Understanding Web Accessibility

Web accessibility means that websites, tools, and technologies are designed and developed so that people with disabilities can use them. More specifically, people can perceive, understand, navigate, and interact with the web.

Accessibility involves a wide range of disabilities, including visual, auditory, physical, speech, cognitive, language, learning, and neurological disabilities. Learn more about WCAG guidelines.

Good accessibility practices also benefit people without disabilities. For example, a person with a temporary injury like a broken arm or someone in a bright environment who cannot read a screen easily.
